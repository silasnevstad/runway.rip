### Set up the project

Follow these steps to get Runway.js up and running on your local machine:

1. Clone the repository and install dependencies:

    ```bash
    `git clone git@github.com:silasnevstad/runway-js.git`
    `cd runway-js`
    `npm install`
    ```

    <Info>
        If you prefer using Yarn, you can replace `npm install` with `yarn install`.
    </Info>
    <br />
2. Rename `.env.example` to `.env.local`
    ```bash
    `mv .env.example .env.local`
    ```

    [//]: # (<Warning>)

[//]: # (    Make sure to update the values in `.env.local` with your own API keys and other sensitive information. Never commit this file to version control.)

[//]: # (</Warning>)
    <br />
3. Open `http://localhost:3000` and done!
