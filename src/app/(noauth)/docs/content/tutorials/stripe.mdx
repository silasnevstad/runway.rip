<div id="overview" />

1. Create a [Stripe](https://stripe.com/login) account.

2. Go to [API Keys](https://dashboard.stripe.com/apikeys) to retrieve your keys, they should look something like this:

     ```plaintext
    pk_live_...
    ```

    ```plaintext
    sk_live_...
    ```

3. Copy them into `.env.local`:

    ```.env.local
    NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_live_...
    NEXT_PUBLIC_STRIPE_SECRET_KEY=sk_live_...
    ```

4. And you're done!
