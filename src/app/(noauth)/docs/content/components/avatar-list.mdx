<p>
    The <InlineHighlight>AvatarList</InlineHighlight> component helps you visually group multiple avatars. You can show them
    in a row or column, stack them with overlap, and even limit how many are displayed (e.g., "+3 more"). Each item is powered
    by the <InlineHighlight type="navigation">[Avatar](/docs/components/avatar)</InlineHighlight> component, so they share all the same properties and fallback behavior.
</p>

<Info>
    You can pass <InlineHighlight>sharedAvatarProps</InlineHighlight> to style every avatar in the list consistently
    (e.g. <InlineHighlight>size="sm"</InlineHighlight> for all).
</Info>

## Props

| **Prop**          | **Type**                        | **Default**           | **Description**                                                                                                     |
|-------------------|---------------------------------|-----------------------|---------------------------------------------------------------------------------------------------------------------|
| avatars           | array of objects                | []                    | Each object can contain <InlineHighlight>src</InlineHighlight>/<InlineHighlight>alt</InlineHighlight> or <InlineHighlight>letter</InlineHighlight>. |
| direction         | `horizontal\|vertical`          | `horizontal`          | Chooses layout direction.                                                                                           |
| overlap           | boolean                         | true                  | If true, applies negative margins (like Slack or GitHub) so the avatars stack.                                      |
| maxVisible        | number                          | 5                     | Max avatars displayed before the last is replaced with <InlineHighlight>+X more</InlineHighlight>.                  |
| sharedAvatarProps | object                          | {}                    | Common props for every <InlineHighlight>Avatar</InlineHighlight> (e.g. <InlineHighlight>size="sm"</InlineHighlight>).            |
| className         | string                          | ''                    | Additional classes appended to the list container.                                                                  |

<br/>
<Info>
    To see how an individual avatar is configured, check the <InlineHighlight type="navigation">[Avatar](/docs/components/avatar)</InlineHighlight> componenent.
</Info>
