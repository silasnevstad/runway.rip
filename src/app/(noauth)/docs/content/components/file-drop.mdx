<p>
    The <InlineHighlight>FileDrop</InlineHighlight> component provides a user-friendly drag-and-drop interface for uploading files.
    It supports customization of border colors, background colors, icons, and text.
</p>

## Props

| **Prop**          | **Type** | **Default**                             | **Description**                                                                                     |
|-------------------|----------|-----------------------------------------|-----------------------------------------------------------------------------------------------------|
| onDrop            | function | -                                       | Callback function that is called when files are dropped. Receives the dropped files as an argument. |
| className         | string   | -                                       | Additional CSS classes for custom styling.                                                          |
| text              | string   | `"Drop files here, or click to select"` | Text displayed inside the drop area.                                                                |
| textColor         | string   | `"text-gray-500"`                       | Text color for the drop area.                                                                       |
| idleBorderColor   | string   | `"border-gray-300"`                     | Border color when the drop area is idle.                                                            |
| activeBorderColor | string   | `"border-blue-500"`                     | Border color when the drop area is active (i.e., when files are dragged over it).                   |
| activeBgColor     | string   | `"bg-blue-100"`                         | Background color when the drop area is active.                                                      |
| icon              | Icon     | `LuFilePlus`                            | Icon component to display inside the drop area.                                                     |
| iconSize          | string   | `"text-5xl"`                            | Size of the icon.                                                                                   |
| borderClass       | string   | `"border-2 border-dashed"`              | CSS classes for the border style of the drop area.                                                  |
