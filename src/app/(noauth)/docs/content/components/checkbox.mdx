The `Checkbox` component provides a customizable checkbox input with various styling options and labeling capabilities.

<Tip>
    Use the `borderRadius="full"` prop to create a circular checkbox.
</Tip>

## Usage

```javascript
import Checkbox from '@/components/atoms/checkbox';

const [hasAgreed, setHasAgreed] = useState(false);

<Checkbox
    id="terms"
    name="terms"
    label="I agree to the terms and conditions"
    checked={isChecked}
    onChange={(e) => setHasAgreed(e.target.checked)}
/>
```

## Props

| Prop          | Type                 | Default     | Description                                                      |
|---------------|----------------------|-------------|------------------------------------------------------------------|
| name          | string               | -           | The name of the checkbox.                                        |
| label         | string               | -           | The label for the checkbox.                                      |
| labelPosition | "left" \| "right"    | `"right"`   | The position of the label relative to the checkbox.              |
| helperText    | string               | -           | Additional helper text for the checkbox.                         |
| size          | "sm" \| "md" \| "lg" | `"md"`      | The size of the checkbox.                                        |
| color         | string               | `"primary"` | The base color (from [tailwind.config](/docs/tutorials/config)). |
| borderRadius  | string               | `"md"`      | The border radius of the checkbox.                               |
| scale         | boolean              | `false`     | If true, gently scales up on hover.                              |
| lift          | boolean              | `false`     | If true, lifts on hover.                                         |
| active        | boolean              | `true`      | If true, the checkbox has an active press-down scale effect.     |
| border        | boolean              | `false`     | If true, adds a border.                                          |
| disabled      | boolean              | `false`     | Whether the checkbox is disabled.                                |
| checked       | boolean              | `false`     | If true, the checkbox is checked.                                |
| onChange      | function             | -           | The change handler for the checkbox.                             |
| className     | string               | `""`        | Additional [Tailwind](https://tailwindcss.com) utility classes.  |
| ...props      | object               | -           | Any other valid HTML attributes for the checkbox.                |

<Info>
    Use the `labelPosition` prop to control the layout of your checkbox and label.
</Info>
