<div>
    The `Accordion` component displays content in collapsible panels.
    It's useful for organizing information without overwhelming users.
</div>

<Info>
    `Accordion` leverages <InlineHighlight type="navigation">[DropdownText](/docs/components/dropdown-text)</InlineHighlight> to manage collapsible items.
</Info>

<br/>
<Divider />
<br/>

<div id="props" />

## Props

| **Name**    | **Type**   | **Default** | **Description**                                                                           |
|-------------|------------|-------------|-------------------------------------------------------------------------------------------|
| className   | string     | ''          | Additional CSS classes applied to the root accordion container                            |
| items       | array      | -           | Array of objects, each containing <InlineHighlight>title</InlineHighlight> & <InlineHighlight>content</InlineHighlight> fields |
| oneOpen     | boolean    | false       | If true, only one accordion panel can be open at once                                     |

<Info>
    The <InlineHighlight>items</InlineHighlight> prop is an array of objects. For each panel, define
    <InlineHighlight>title</InlineHighlight> (string) and <InlineHighlight>content</InlineHighlight> (JSX or string).
</Info>

<Tip>
    <strong>Pro Tip:</strong> Try nesting an Accordion inside another Accordion to create a multi-level collapsible structure for more complex documentation.
</Tip>

### 3. Nested Accordion
Example of a nested structure: each "item" could itself contain an Accordion.

<PreviewCode preview={
    <div className="flex w-full px-2">
        <Accordion
            items={[
                {
                    title: "Nested Accordion",
                    // Optionally include some content before the nested accordion:
                    content: "Content above the nested accordion:",
                    // The nested items are defined under `items`
                    items: [
                        {
                            title: "Nested Item 1",
                            content: "Content for nested item 1.",
                        },
                        {
                            title: "Nested Item 2",
                            // You can nest arbitrarily deep:
                            items: [
                                {
                                    title: "Grandchild Item",
                                    content: "Content for grandchild item.",
                                },
                            ],
                        },
                    ],
                },
                {
                    title: "Simple Item",
                    content: "This is a simple accordion item.",
                },
            ]}
        />
    </div>
}>
    ```javascript
    <Accordion
        items={[
            {
                title: 'Parent Item 1',
                content: (
                    <Accordion
                        items={[
                            {
                                title: 'Child Item A',
                                content: 'Content of Child A.'
                            },
                            {
                                title: 'Child Item B',
                                content: 'Content of Child B.'
                            }
                        ]}
                    />
                )
            },
            {
                title: 'Parent Item 2',
                content: 'This is the second parent item content.'
            }
        ]}
    />
    ```
</PreviewCode>

<br/>
<Caution>
    Be mindful of nesting too many layers of accordions. Deep nesting can confuse users or clutter your layout.
</Caution>
