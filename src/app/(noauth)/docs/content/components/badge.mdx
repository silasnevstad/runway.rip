The <InlineHighlight>Badge</InlineHighlight> component is a minimal, inline element. It's often used to display short, contextual text such as statuses, labels, or counts. You can customize its shape, color, style, and even add a hover effect or click handler.

<Tip>
    For best results, keep your badge text short—one or two words or small numbers.
</Tip>

## Props

| **Name**    | **Type**                  | **Default** | **Description**                                                                                         |
|-------------|---------------------------|-------------|---------------------------------------------------------------------------------------------------------|
| children    | ReactNode                 | -           | Content inside the badge (usually text or an icon).                                                     |
| className   | string                    | ''          | Additional utility classes appended to the badge.                                                       |
| shape       | 'pill', 'square', 'circle' | 'pill'    | Determines the badge’s rounded corners and padding.                                                     |
| border      | boolean                  | false       | If true, adds a border around the badge (color depends on the badge’s style/color).                     |
| style       | 'soft', 'solid'          | 'soft'      | soft shows a tinted background, solid is fully colored with white text.                          |
| color       | <span className="text-primary-500">'primary'</span>, <span className="text-green-500">'green'</span>, <span className="text-red-500">'red'</span>, <span className="text-yellow-500">'yellow'</span>, <span className="text-orange-500">'orange'</span>, <span className="text-blue-500">'blue'</span>, <span className="text-purple-500">'purple'</span>, <span className="text-gray-500">'gray'</span> | <span className="text-primary-500">'primary'</span> | The badge’s color scheme. Combines with style to determine background/text color.                       |
| hover       | boolean                 | false       | If true, the badge slightly lifts (translate-y) on hover (for a subtle interactive feel).               |
| onClick     | Function                | -           | Callback fired when the badge is clicked.                                                               |

<Info>
    You can combine <InlineHighlight>hover</InlineHighlight> and <InlineHighlight>onClick</InlineHighlight> to make your badge feel like a button.
</Info>

<Tip>
    Combining badges with tooltips can be a great way to expand context without cluttering the UI.
</Tip>
