<div>
    The `Badge` component is a simple yet versatile element for labeling, categorizing, or drawing attention to items in your interface. It comes with flexible `size` options, `color` themes, and allows you to tweak the `borderRadius` for shaping.
</div>

<Info>
    The `Badge` component is great for displaying information that is not critical but still important.
</Info>

## Usage

```javascript
import Badge from '@/components/atoms/Badge';

<Badge color="primary" variant="soft">
    Hello Badge
</Badge>
```

## Props

| **Prop**     | **Type**                     | **Default** | **Description**                                                                                                       |
|--------------|------------------------------|-------------|-----------------------------------------------------------------------------------------------------------------------|
| children     | React.ReactNode              | -           | Content displayed inside the badge                                                                                    |
| className    | string                       | `""`        | Additional [Tailwind](https://tailwindcss.com) classes (overrides defaults if needed)                                 |
| borderRadius | string                       | `"full"`    | The tailwind-friendly [rounding for corners](https://tailwindcss.com/docs/border-radius) (e.g. `"md"`, `"full"`, etc) |
| size         | "xs" \| "sm" \| "md" \| "lg" | `"md"`      | Determines overall size of the badge                                                                                  |
| variant      | "soft" \| "solid"            | `"soft"`    | Selects a muted (`soft`) or bold (`solid`) visual style                                                               |
| color        | string                       | `"primary"` | Base color. Must match your Tailwind color keys (e.g. `primary`, `gray`, `blue`)                                      |
| border       | boolean                      | `false`     | Adds a border in a complementary color                                                                                |
| hover        | boolean                      | `false`     | Enables a hover animation that gently lifts the badge                                                                 |
| onClick      | function                     | -           | If provided, the badge is clickable; includes a small press-scale animation                                           |
| ...props     | object                       | -           | Any other props you’d like to pass to the root `<div>`                                                                |

## Examples

### 1. Basic Badges
A swift example with minimal configuration. Just drop in some content!

<PreviewCode preview={
    <div className="flex gap-3">
        <Badge>Default</Badge>
        <Badge color="orange" variant="solid">Solid Orange</Badge>
        <Badge color="red" border>Soft Red</Badge>
    </div>
}>
    ```javascript
    <div className="flex gap-3">
        <Badge>Default</Badge>
        <Badge color="orange" variant="solid">Solid Orange</Badge>
        <Badge color="red" border>Soft Red</Badge>
    </div>
    ```
</PreviewCode>

<br/>

### 2. Different Shapes (borderRadius)
Use `borderRadius` to shape corners—like `"none"`, `"md"`, `"full"` (circle-like).

<PreviewCode preview={
    <div className="flex gap-3">
        <Badge borderRadius="none">None</Badge>
        <Badge borderRadius="lg">Rounded</Badge>
        <Badge borderRadius="full" variant="solid">Full</Badge>
    </div>
}>
    ```javascript
    <div className="flex gap-3">
        <Badge borderRadius="none">None</Badge>
        <Badge borderRadius="lg">Rounded</Badge>
        <Badge borderRadius="full">Full</Badge>
    </div>
    ```
</PreviewCode>

<Info>
    Tailwind’s [“rounded”](https://tailwindcss.com/docs/border-radius) classes come in a bunch of variations. Pass them in as you like—maybe <code>borderRadius="xl"</code> for super smooth corners!
</Info>

<br/>

### 3. Adding Hover and Custom Classes
Spice it up your own [Tailwind](https://tailwindcss.com) classes and a subtle hover animation. Great for clickable or important badges.

<PreviewCode preview={
    <div className="flex gap-3">
        <Badge lift>
            Lift on Hover
        </Badge>
        <Badge scale>
            Scale on Hover
        </Badge>
        <Badge className={"bg-blue-500 dark:bg-blue-500 text-bg-0  dark:text-bg-900"}>Custom</Badge>
    </div>
}>
    ```javascript
    <div className="flex gap-3">
        <Badge lift>
            Lift on Hover
        </Badge>
        <Badge scale>
            Scale on Hover
        </Badge>
        <Badge className={
            "bg-blue-500 dark:bg-blue-500 text-bg-0  dark:text-bg-900"
        }>
            Custom
        </Badge>
    </div>
    ```
</PreviewCode>

<Tip>
    If you need your own styles, just add them via <code>className</code>.
</Tip>
