## Props

| **Prop**          | **Type** | **Default** | **Description**                                                                                           |
|-------------------|----------|-------------|-----------------------------------------------------------------------------------------------------------|
| sections          | array    | `[]`        | Breadcrumb route definitions, e.g. `{ href, title, items?: nested }`.                                     |
| delimiter         | Icon     | `<Icon />`  | Separator between crumbs.                                                                                 |
| isInteractiveLast | boolean  | `false`     | Makes the final crumb clickable if `true`.                                                                |
| className         | string   | `""`        | Extra [Tailwind](https://tailwindcss.com) classes. These override internal defaults using `mergeClasses`. |
| ...props          | object   | `-`         | Additional props (e.g. `style`, `onMouseEnter`) passed down to the main `<nav>` element.                  |

## Examples

### Default Breadcrumb
Just drop it in and it will automatically parse the current path.

<PreviewCode
    preview={
        <Breadcrumb />
    }
>
    ```javascript
    <Breadcrumb />
    ```
</PreviewCode>

<br />

### Provide Sections
Create an array of objects with `href` and `title` properties. Breadcrumb will automatically parse the current path
and highlight the matching crumb.

<PreviewCode
    preview={
        <Breadcrumb
            sections={[
                {
                    href: "/docs",
                    title: "Docs",
                    items: [
                        {
                            href: "/docs/components",
                            title: "Components",
                            items: [
                                { href: "/docs/components/breadcrumb", title: "Breadcrumb" },
                            ]
                        }
                    ]
                }
            ]}
        />
    }
>
    ```javascript
    <Breadcrumb
        sections={[
            {
                href: "/docs",
                title: "Docs",
                items: [{
                    href: "/docs/components",
                    title: "Components",
                    items: [{
                        href: "/docs/components/breadcrumb",
                        title: "Breadcrumb"
                    }]
                }]
            }
        ]}
    />
    ```
</PreviewCode>

<Tip>
    Make sure each item has a `title` and `href` property.
</Tip>

<br />

### Custom Delimiter

Swap in any delimiter you want (like a slash `/` or text `>`).

<PreviewCode
    preview={
        <Breadcrumb
            sections={[]}
            delimiter="/"
        />
    }
>
    ```javascript
    <Breadcrumb
        sections={[]}
        delimiter="/"
    />
    ```
</PreviewCode>
