## Props

| **Name**   | **Type**  | **Default** | **Description**                                                                                           |
|------------|-----------|-------------|-----------------------------------------------------------------------------------------------------------|
| sections   | `Array`   | -           | An array of objects representing each section of the breadcrumb. Each object has a <code>href</code> and <code>title</code>. The optional <code>items</code> array can nest more sections. |
| delimiter  | ReactNode | `FaChevronRight` | (Optional) Icon or text inserted between breadcrumb items.                                                |
| isInteractiveLast | boolean | `false`  | (Optional) If <code>false</code>, the final breadcrumb is a non-clickable <code>&lt;span&gt;</code> instead of a link. |
