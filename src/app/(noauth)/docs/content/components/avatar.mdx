<div>
    The `Avatar` component displays a user’s profile image. If the image fails to load or isn’t provided, it
    automatically falls back to a letter. It supports various `sizes`, `shapes`, `colors`, and `status` indicators.
</div>

<Info>
    For multiple avatars, see the
    <InlineHighlight type="navigation">[AvatarList](/docs/components/avatar-list)</InlineHighlight> component.
</Info>

## Usage

```javascript
import Avatar from '@/components/atoms/avatar';

<Avatar src="/avatar1.png" alt="User Image"/>
```

## Props

| **Prop**     | **Type**                 | **Default**    | **Description**                         |
|--------------|--------------------------|----------------|-----------------------------------------|
| src          | string                   | -              | Image URL                               |
| alt          | string                   | ''             | Image alt text                          |
| letter       | string                   | -              | Letter to display                       |
| color        | string                   | `gray`         | Avatar color (from tailwind.config)     |
| size         | xs, sm, md, lg, xl       | `md`           | Avatar size                             |
| borderRadius | xs, sm, md, lg, xl, full | `full`         | Border radius (full for circle)         |
| status       | string                   | -              | A status indicator (e.g. 'online')      |
| statusColor  | string                   | `bg-green-400` | Status color (from tailwind.config)     |
| className    | string                   | ''             | Additional classes (overrides defaults) |
| onClick      | function                 | -              | Click handler                           |
| **...props** | object                   | -              | Additional props for the image element  |

<br/>
<Tip>
    Feel free to customize the `status` prop any way you like, such as 'online' or 'busy'.
</Tip>

Simply provide a `src` and `alt` prop. Optionally, style with other props.

### 2. Letter
If you want a letter instead of an image, just provide the `letter` prop.

<PreviewCode preview={
    <div className="flex gap-3">
        <Avatar
            letter="Z"
            borderRadius="sm"
        />
        <Avatar
            alt="Ava Jones"
            src="/invalid-img.jpg"
        />
    </div>
}>
    ```javascript
    <div className="flex gap-3">
        <Avatar
            letter="Z"
            borderRadius="sm"
        />
        <Avatar
            alt="Ava Jones"
            src="/invalid-img.jpg"  // this will fail
        />
    </div>
    ```
</PreviewCode>

<Info>
    If the image is missing or broken, `Avatar` shows the the `letter` or the first character of the `alt` text.
</Info>

<br />
