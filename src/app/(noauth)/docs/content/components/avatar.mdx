## Avatar

The `Avatar` component displays an image avatar using Next.js's `Image` component for optimal performance.

<PreviewCode preview={
    <Avatar src="/images/elon-avatar.png" alt="Elon Musk" />
}>
    ```jsx
    <Avatar src="/public/images/user.jpg" alt="User Name" />
    ```
</PreviewCode>

<Tip>
    Use the `width` and `height` props to adjust the size of the avatar. The default size is 50x50 pixels.
</Tip>
<Warning>
    Ensure that the `src` prop points to a valid image URL. Invalid URLs may cause rendering issues.
</Warning>

## Letter Avatar

The `LetterAvatar` component displays a single letter, typically the user's initial, when an image is not available.

<PreviewCode preview={
    <LetterAvatar letter="J" />
}>
    ```jsx
    <LetterAvatar letter="J" />
    ```
</PreviewCode>

<Info>
    The `LetterAvatar` is useful for representing users who haven't uploaded a profile picture or when you want to display initials instead of images.
</Info>
<Tip>
    You can customize the background color and text color by modifying the className prop to match your application's color scheme.
</Tip>

## Avatar List

The `AvatarList` component displays a group of avatars, typically used to show multiple users or participants.

<PreviewCode preview={
    <AvatarList avatars={[
        { letter: "A" },
        { letter: "B" },
        { letter: "C" },
    ]} />
}>
    ```jsx
    <AvatarList avatars={[
        { letter: "A" },
        { letter: "B" },
        { letter: "C" },
    ]} />
    ```
</PreviewCode>

<Info>
    The `AvatarList` component slightly overlaps avatars to create a compact group representation.
</Info>
<Tip>
    Limit the number of avatars displayed to prevent overcrowding. Consider showing a "+X more" indicator for additional users.
</Tip>
