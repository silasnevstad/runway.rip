<p>
    The <InlineHighlight>Avatar</InlineHighlight> component displays a user’s profile image if available.
    When the image fails to load or isn’t provided, it falls back to a letter. You can customize <InlineHighlight>size</InlineHighlight>, <InlineHighlight>shape</InlineHighlight>, ring colors,
    status indicators, and more.
</p>

<Info>
    For collections of multiple avatars, see the
    <InlineHighlight type="navigation">[AvatarList](/docs/components/avatar-list)</InlineHighlight> component.
</Info>

## Props

| **Prop**       | **Type**                         | **Default**                  | **Description**                                                                                                                              |
|----------------|----------------------------------|------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------|
| src            | string                           | -                            | URL of the image. If absent or fails to load, we show a fallback letter.                                                                     |
| alt            | string                           | ''                           | Accessible text (for screen readers) and used for the fallback letter.                                                                       |
| letter         | string                           | -                            | Explicit letter to show if no image (overrides the first character of <InlineHighlight>alt</InlineHighlight>).                               |
| size           | `xs\|sm\|md\|lg\|xl`             | `md`                         | Controls the Avatar dimensions (width/height/font size).                                                                                     |
| shape          | `circle\|rounded\|square`        | `circle`                     | Determines corner rounding.                                                                                                                  |
| borderColor    | string                           | `gray-400 dark:border-gray-500` | Tailwind classes for the outer border color.                                                                                              |
| bgColor        | string                           | `bg-gray-200 dark:bg-gray-700`  | Tailwind classes for the avatar background.                                                                                                |
| textColor      | string                           | `text-gray-600 dark:text-gray-300` | Tailwind classes for fallback letter text color.                                                                                         |
| ringColor      | string                           | -                            | Tailwind classes for ring highlight, e.g. <InlineHighlight>ring-2 ring-blue-500</InlineHighlight>.                                          |
| status         | string                           | -                            | Status label (e.g. <InlineHighlight>"online"</InlineHighlight>). If provided, we show a small colored dot in the bottom-right corner.       |
| statusColor    | string                           | `bg-green-400`               | Tailwind class for the status dot color.                                                                                                    |
| className      | string                           | ''                           | Additional classes appended to the outer container.                                                                                         |
| onClick        | function                         | -                            | Callback fired when the avatar is clicked.                                                                                                  |
| style          | object                           | -                            | Inline style object for custom styling.                                                                                                     |

<br/>
<Info>
    For advanced usage, you can also add the <InlineHighlight>ringColor</InlineHighlight> prop to highlight the avatar with a Tailwind ring (e.g. <InlineHighlight>ring-2 ring-primary-500</InlineHighlight>).
</Info>

### 2. Letter Fallback
If an image is missing or broken, <InlineHighlight>Avatar</InlineHighlight> shows the first character of <InlineHighlight>alt</InlineHighlight> or the explicit <InlineHighlight>letter</InlineHighlight> prop.

<PreviewCode preview={
    <div className="flex gap-3">
        <Avatar
            alt="Ava Jones"
            src="/invalid-img.jpg"
        />
        <Avatar
            letter="Z"
            shape="square"
        />
    </div>
}>
    ```javascript
    <div className="flex gap-3">
        <Avatar
            alt="Ava Jones"
            src="/invalid-img.jpg"  // this will fail
        />
        <Avatar
            letter="Z"
            shape="square"
        />
    </div>
    ```
</PreviewCode>
