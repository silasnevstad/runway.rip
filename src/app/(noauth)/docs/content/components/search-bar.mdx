<p>
    The <InlineHighlight>SearchBar</InlineHighlight> component allows users to search through a list of items that are organized into sections. The component filters both section titles and sub-items based on the search query and displays the results in a dropdown.
</p>

<Info>
    It leverages the <InlineHighlight>Input</InlineHighlight> atom for the input field and the <InlineHighlight>mergeClasses</InlineHighlight> utility for optimal styling. This component is designed to be easily integrated into documentation pages or other parts of your application.
</Info>

## Props

| **Prop**           | **Type**    | **Default**    | **Description**                                                                                                     |
|--------------------|-------------|----------------|---------------------------------------------------------------------------------------------------------------------|
| items              | array       | -              | An array of section objects. Each section should have a `title` and an `items` array with sub-items.                |
| onItemSelect       | function    | -              | Callback fired when an item is selected. Receives the selected item as an argument.                                  |
| renderItem         | function    | -              | Optional. A function to customize how each item is rendered.                                                        |
| containerClassName | string      | `""`           | Additional classes for the outer container of the SearchBar.                                                        |
| inputClassName     | string      | `""`           | Additional classes for the Input component.                                                                         |
| placeholder        | string      | `"Search..."`  | Placeholder text for the search input.                                                                              |
