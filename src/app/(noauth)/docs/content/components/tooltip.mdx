<p>
    The <InlineHighlight>Tooltip</InlineHighlight> component provides a lightweight and customizable tooltip that appears when hovering over an element. It supports various positions and custom styling through Tailwind CSS classes.
</p>

<Info>
    This component leverages the <InlineHighlight>mergeClasses</InlineHighlight> utility to seamlessly combine default classes with any custom class names passed via props.
</Info>

<br/>
<Divider />
<br/>

<div id="props" />

## Props

| **Prop**   | **Type**    | **Default** | **Description**                                                               |
|------------|-------------|-------------|-------------------------------------------------------------------------------|
| children   | ReactNode   | -           | The element that triggers the tooltip on hover.                             |
| text       | string      | -           | The text content displayed within the tooltip.                              |
| className  | string      | `""`        | Additional CSS classes for the tooltip container.                           |
| position   | string      | `"top"`     | Position of the tooltip relative to the children (`top`, `bottom`, `left`, or `right`). |
