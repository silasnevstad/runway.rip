<p>
    The <InlineHighlight>Switch</InlineHighlight> component provides a segmented control interface
    for toggling between multiple options. It supports both horizontal and vertical layouts,
    customizable colors, hover effects, and optional tooltips.
</p>

## Props

| **Prop**         | **Type**            | **Default**              | **Description**                                                                                                   |
|------------------|---------------------|--------------------------|-------------------------------------------------------------------------------------------------------------------|
| options          | array               | -                        | An array of option objects. Each option may contain: `value`, `name`, `Icon`, or `image`.                         |
| selected         | any                 | -                        | The currently selected option value.                                                                              |
| onChange         | function            | -                        | Callback fired when an option is selected. Receives the selected option's value.                                    |
| activeColor      | string              | `"blue"`                 | Text color for the active option (e.g., `"blue"`, `"red"`).                                                       |
| inactiveColor    | string              | `"gray"`                 | Text color for inactive options.                                                                                  |
| activeBgColor    | string              | `"blue"`                 | Background color for the active option.                                                                           |
| inactiveBgColor  | string              | `"gray"`                 | Background color for inactive options.                                                                            |
| hover            | boolean             | `false`                  | If `true`, applies a hover effect on each option.                                                                 |
| tooltip          | boolean             | `false`                  | If `true`, shows a tooltip with the option name on hover.                                                         |
| vertical         | boolean             | `false`                  | If `true`, displays options in a vertical layout.                                                                 |
| shape            | string              | `"rounded-full"`         | Border radius style for both the container and each option button.                                                |
| className        | string              | `""`                     | Additional CSS classes for custom styling of the container.                                                       |
