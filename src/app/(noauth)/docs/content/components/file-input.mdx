    It features a customizable button with an icon and displays the selected file name.
</p>

## Props

| **Prop**          | **Type**   | **Default**                   | **Description**                                                                                           |
|-------------------|------------|-------------------------------|-----------------------------------------------------------------------------------------------------------|
| onFileChange      | function   | -                             | Callback function that is called when a file is selected. Receives the selected file as an argument.      |
| text              | string     | `"Choose File"`               | Text displayed on the file input button.                                                                  |
| placeholder       | string     | `"No file selected"`          | Placeholder text displayed when no file is selected.                                                      |
| className         | string     | -                             | Additional CSS classes for custom styling.                                                                |
| buttonClassName   | string     | -                             | Additional CSS classes for the file input button.                                                         |
| inputClassName    | string     | -                             | Additional CSS classes for the file input element.                                                        |
| color             | string     | `"gray"`                      | Color scheme for the file input button.                                                                   |
| buttonIcon        | Icon       | `FiFilePlus`                  | Icon component to display on the file input button.                                                       |
