## Props

| **Prop**             | **Type**         | **Default**              | **Description**                                                                                                         |
|----------------------|------------------|--------------------------|-------------------------------------------------------------------------------------------------------------------------|
| children             | ReactNode             | -                        | The content of the button.                                                                                             |
| onClick              | function         | -                        | Click event handler.                                                                                                   |
| href                 | string           | -                        | If provided, the button will render as a Next.js <InlineHighlight type="code">Link</InlineHighlight>.                   |
| className            | string           | -                        | Additional CSS classes for custom styling.                                                                             |
| disabled             | boolean          | `false`                  | If `true`, the button will be disabled.                                                                                 |
| loading              | boolean          | `false`                  | If `true`, shows a loading spinner and disables pointer events.                                                          |
| shadow               | boolean          | `false`                  | If `true`, applies a shadow effect to the button.                                                                       |
| active               | boolean          | `true`                   | If `true`, applies an active effect when the button is pressed.                                                         |
| grow                 | boolean          | `false`                  | If `true`, the button will grow to fill the width of its container.                                                      |
| lift                 | boolean          | `false`                  | If `true`, applies a lift effect to the button.                                                                         |
| icon                 | Icon             | -                        | A React component to render as an icon inside the button.                                                               |
| iconSrc             | string           | -                        | URL of an external image to render as an icon inside the button.                                                         |
| iconAlt             | string           | -                        | Alt text for the icon image.                                                                                            |
| iconClassname        | string           | -                        | Additional CSS classes for the icon.                                                                                   |
| textColor            | string           | -                        | Tailwind CSS classes for text color.                                                                                    |
| backgroundColor      | string           | -                        | Tailwind CSS classes for background color.                                                                              |
| hoverBackgroundColor | string           | -                        | Tailwind CSS classes for background color on hover.                                                                     |
| shape                | string           | -                        | Tailwind CSS classes for border radius.                                                                                 |

<br/>
<Info>
    <strong>Note:</strong> If both <InlineHighlight>icon</InlineHighlight> and <InlineHighlight>iconSrc</InlineHighlight> are provided, <InlineHighlight>icon</InlineHighlight> (the React component) takes precedence.
</Info>

## Examples

### 1. Basic Usage

<PreviewCode preview={
    <div className="flex flex-wrap gap-4">

    ```javascript
        <Button className="bg-green-500 hover:bg-green-600">Success</Button>
    </div>
    ```

<br />

### 2. Disabled & Loading
When <InlineHighlight>disabled</InlineHighlight> or <InlineHighlight>loading</InlineHighlight> is set, the button
cannot be clicked. <InlineHighlight>loading</InlineHighlight> also shows a <InlineHighlight>Loader</InlineHighlight>.

<PreviewCode preview={
    <div className="flex flex-wrap gap-4">
        <Button disabled>Disabled</Button>
        <Button loading variant="warning">
            Loading...
        </Button>
    </div>
}>
    ```javascript
    <div className="flex flex-wrap gap-4">
        <Button disabled>Disabled</Button>
        <Button loading variant="warning">
            Loading...
        </Button>
    </div>
    ```
</PreviewCode>

<br />

### 3. Icons
Pass a React icon component (<InlineHighlight>icon</InlineHighlight>) or an external image (<InlineHighlight>iconSrc</InlineHighlight>).

<PreviewCode preview={
    <ButtonIconDemo />
}>
    ```javascript
    <div className="flex flex-wrap gap-4 items-center">
        <Button
            icon={() => <ClipboardIcon className="w-5 h-5" />}
            className="px-5"
        >
            Copy
        </Button>
        <Button
            icon={() => <InformationCircleIcon className="w-5 h-5" />}
        >
            Click Me
        </Button>
    </div>
    ```
</PreviewCode>

<br />

### 4. As a Link
If you pass an <InlineHighlight>href</InlineHighlight> prop, the button renders as a Next.js <InlineHighlight>Link</InlineHighlight>.

<PreviewCode preview={
    <div className="flex flex-wrap gap-4">
        <Button href="/docs" variant="underline">
            Go to Docs
        </Button>
    </div>
}>
    ```javascript
    <div className="flex flex-wrap gap-4">
        <Button href="/docs" variant="underline">
            Go to Docs
        </Button>
    </div>
    ```
</PreviewCode>

<br />
<Tip>
    Combine <InlineHighlight>href</InlineHighlight> with a <InlineHighlight>target="_blank"</InlineHighlight>
    for external links if desired.
</Tip>

<br />
<Caution>
    <strong>Warning:</strong> Avoid using <InlineHighlight>href</InlineHighlight> and <InlineHighlight>onClick</InlineHighlight>
    together if they lead to conflicting actions. Typically <InlineHighlight>href</InlineHighlight> takes precedence.
</Caution>
