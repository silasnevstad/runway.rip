## Usage

```javascript
import Alert from '@/components/atoms/Alert';

<Alert severity="info">This is an info message.</Alert>
```

## Props

| **Prop**  | **Type**                                    | **Default** | **Description**                                                                       |
|-----------|---------------------------------------------|-------------|---------------------------------------------------------------------------------------|
| children  | ReactNode                                   | -           | Content to be displayed inside the alert component                                    |
| severity  | "info" \| "success" \| "warning" \| "error" | `"info"`    | Severity level of the alert component (info, success, warning, error)                 |
| variant   | "soft" \| "solid"                           | `"soft"`    | Variant style of the alert component                                                  |
| lift      | boolean                                     | `false`     | Whether to lift the alert component on hover                                          |
| scale     | boolean                                     | `false`     | If true, gently scales up on hover.                                                   |
| lift      | boolean                                     | `false`     | If true, lifts on hover.                                                              |
| active    | boolean                                     | `true`      | If true, the button has an active press-down scale effect.                            |
| border    | boolean                                     | `false`     | Whether to show a border around the alert component                                   |
| onClick   | function                                    | -           | Function to call when the alert is clicked                                            |
| className | string                                      | `""`        | Additional [Tailwind](https://tailwindcss.com) classes (overrides defaults if needed) |
| ...props  | object                                      | -           | Additional props passed to the alert component                                        |

## Examples

### 1. Basic Alert
Just provide a `severity` prop.

<PreviewCode preview={
    <div className="flex flex-col gap-3">
        <Alert>This is an info message.</Alert>
        <Alert severity="success">This is a success message.</Alert>
        <Alert severity="warning">This is a warning message.</Alert>
        <Alert severity="error">This is a danger message.</Alert>
    </div>
}>

    ```javascript
    <Alert>This is an info message.</Alert>
    <Alert severity="success">This is a success message.</Alert>
    <Alert severity="warning">This is a warning message.</Alert>
    <Alert severity="error">This is a danger message.</Alert>
    ```
</PreviewCode>

<Tip>
    Make `Alert` clickable by adding an `onClick` prop.
</Tip>

### 2. Styled Alert
You can customize the alert with variants, borders and additional [Tailwind](https://tailwindcss.com) classes.

<PreviewCode preview={
    <div className="flex flex-col gap-3">
        <Alert severity="success" variant="soft" border>This is a soft alert with a border.</Alert>
        <Alert variant="soft" className="bg-primary-500/40 dark:bg-primary-500/20">This is a soft alert with custom background color.</Alert>
    </div>
}>

    ```javascript
    <Alert variant="solid" border>This is a solid alert with a border.</Alert>
    <Alert variant="soft" className="bg-primary-500/40 dark:bg-primary-500/20">This is a soft alert with custom background color.</Alert>
    ```
</PreviewCode>


### 3. Hover Alert
You can add a hover effect to the alert by setting the `lift` or `scale` prop.

<PreviewCode preview={
    <div className="flex flex-col gap-3">
        <Alert severity="info" lift>This is an info message.</Alert>
        <Alert severity="success" scale>This is a success message.</Alert>
        <Alert severity="warning" lift scale>This is a warning message.</Alert>
    </div>
}>

    ```javascript
    <Alert severity="info" lift>This is an info message.</Alert>
    <Alert severity="success" scale>This is a success message.</Alert>
    <Alert severity="warning" lift scale>This is a warning message.</Alert>
    ```
</PreviewCode>
