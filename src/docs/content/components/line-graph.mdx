<div>
    The `LineGraph` component is a versatile charting solution that renders a line graph using Chart.js. It allows you to customize the line curve, point markers, axis styling, fill opacity, and more – making it ideal for financial data, performance charts, or any time-series visualization.
</div>

<Info>
    The component uses [react-chartjs-2]() and [Chart.js]() under the hood.
</Info>

## Props

| **Prop**            | **Type** | **Default**                                                                                                           | **Description**                                                                                                                           |
|---------------------|----------|-----------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| data                | object   | -                                                                                                                     | The chart data, including `labels` and `datasets`. Each dataset should have a `data` array (and optionally `x` values) for plotting.      |
| curved              | boolean  | `true`                                                                                                                | If `true`, the line is rendered with a curve (default tension of 0.3). If `false`, the line is straight (tension = 0).                    |
| showPoints          | boolean  | `true`                                                                                                                | Toggle to display data point markers on the graph.                                                                                        |
| lastPointMarkerOnly | boolean  | `false`                                                                                                               | If `true`, only the last data point is marked (ideal for stock charts with a status indicator).                                           |
| markerOptions       | object   | `{ radius: 3, pointBackgroundColor: "#6b72fdfc", pointBorderColor: "#6b72fdfc", pointBorderWidth: 2, lastRadius: 6 }` | Customization options for point markers. You can define default marker styles and override the last point styling with `lastRadius`, etc. |
| axisStyles          | object   | `{}`                                                                                                                  | Custom styles for the x and y axes. Can include settings for ticks, grid lines, and title styling.                                        |
| lineStyle           | object   | `{}`                                                                                                                  | Custom styles for the line element. Override properties such as `tension`, `fill`, `backgroundColor`, `borderColor`, and `borderWidth`.   |
| fillOpacity         | number   | `0.4`                                                                                                                 | Opacity for the line fill when `fill` is enabled.                                                                                         |
| xAxisLabel          | string   | `""`                                                                                                                  | Label for the x-axis.                                                                                                                     |
| yAxisLabel          | string   | `""`                                                                                                                  | Label for the y-axis.                                                                                                                     |
| customOptions       | object   | `{}`                                                                                                                  | Additional Chart.js options to merge with the default options.                                                                            |

<br/>
<Info>
    For advanced customization, you can use the <InlineHighlight>lineStyle</InlineHighlight> and <InlineHighlight>customOptions</InlineHighlight> props to further tweak the graph’s appearance.
</Info>

### 2. Straight Line with Only the Last Point Marked

<PreviewCode preview={
    <LineGraph
        data={{

                label: 'Profit',

                borderColor: '#ff5733'
