The `DropdownItem` component is a dropdown item that can be used to display hidden content. It is a great way to organize and display information in a structured and visually appealing way.

<Callout>
    Want to use multiple dropdowns? Check out the [`Accordion`](/docs/components/accordion) component.
</Callout>

## Usage

```js
import DropdownItem from '@/components/atoms/DropdownItem';

<DropdownItem
    header="Dropdown Text"
>
    This is the content of the dropdown text component.
</DropdownItem>
```

<Tip>
    Use `borderStyle` to change the border style of the dropdown text component.
</Tip>

## Props

The `DropdownItem` component accepts the following props:

| Name        | Type                      | Default     | Description                                                                                                                  |
|-------------|---------------------------|-------------|------------------------------------------------------------------------------------------------------------------------------|
| header      | string                    | -           | The title of the dropdown text component.                                                                                    |
| children    | string                    | -           | The content of the dropdown text component.                                                                                  |
| initialOpen | boolean                   | false       | If true, the dropdown text component is open by default.                                                                     |
| isOpen      | boolean                   | false       | If true, the dropdown text component is open.                                                                                |
| onToggle    | function                  | -           | A function that is called when the dropdown text component is toggled.                                                       |
| onClick     | function                  | -           | A function that is called when the dropdown text component is clicked.                                                       |
| color       | string                    | `"gray"`    | The base color (uses [tailwind.config](/docs/features/config#color-theme)).                                                  |
| activeColor | string                    | `"primary"` | The base color (uses [tailwind.config](/docs/features/config#color-theme)).                                                  |
| variant     | "soft" \| "solid"         | `"soft"`    | The variant of the component.                                                                                                |
| hoverBg     | boolean                   | false       | If true, the dropdown text component has a hover background color.                                                           |
| border      | boolean                   | true        | If true, the dropdown text component has a border.                                                                           |
| borderStyle | "b" \| "t" \| "l" \| "r"  | `"b"`       | The border style of the dropdown text component (uses [Tailwind](https://tailwindcss.com/docs/border-width) border classes). |
| iconType    | "chevron" \| "plus-minus" | `"chevron"` | The icon type of the dropdown text component.                                                                                |
| className   | string                    | `""`        | Additional [Tailwind](https://tailwindcss.com) utility classes.                                                              |
| ...props    | object                    | -           | Any other valid HTML attributes.                                                                                             |

<Info>
    The component can either manage its own state or be controlled by the parent component using the `isOpen` and `onToggle` props.
</Info>
