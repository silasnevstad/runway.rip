<br />

The `AuthInput` component is essentially just a wrapper around the [`Input`](/docs/components/atoms/input) component. It's
useful for differentiating between different types of inputs in an authentication form and for showing errors if you don't
want [`ToastProvider`](/docs/components/toast-provider) to be used.

## Usage

```javascript
import AuthInput from '@/components/auth/AuthInput';

<AuthInput
    label="Email"
    name="email"
    type="email"
/>
```

## Props

| Name        | Type    | Default | Description                                                                 |
|-------------|---------|---------|-----------------------------------------------------------------------------|
| error       | string  | `""`    | The error message to display.                                               |

<Info>
    Besides the props listed above, it also accepts all the props of the [`Input`](/docs/components/atoms/input) component.
</Info>
