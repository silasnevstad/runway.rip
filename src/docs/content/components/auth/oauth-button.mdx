<div id="overview" />

The `OAuthButton` component is a button component that can be used to display an OAuth button. It uses the [`Button`](/docs/components/button) component for the button and `/public/logos` for the logos.

<br />
<br />
<Divider />
<div id="usage" />
## Usage

```javascript
import OAuthButton from '@/components/auth/OAuthButton';

<OAuthButton />
```

<Info>
    [`AuthForm`](/docs/components/auth/auth-form) takes care of all this automatically and uses the `OAuthButton` component to display the OAuth buttons.
</Info>

<div id="props" />
## Props

| Name     | Type   | Default | Description                                                            |
|----------|--------|---------|------------------------------------------------------------------------|
| provider | string | `""`    | The provider of the OAuth button (one of `google`, `github`, `apple`). |
| logoSrc  | string | `""`    | The src of the logo.                                                   |
| logoAlt  | string | `""`    | The alt of the logo.                                                   |
| onClick  | func   | `null`  | The onClick handler of the button.                                     |
