The `AccountCard` component is a card with a dropdown menu that displays account information. It's customizable and let's
users manage their account settings, billing, and logout.

<br />

<Info>
    It uses [`Button`](/docs/components/button) and [`Avatar`](/docs/components/avatar) components for the dropdown menu and avatar respectively.
</Info>

## Props

| Name            | Type    | Default | Description                                                                        |
|-----------------|---------|---------|------------------------------------------------------------------------------------|
| size            | string  | `md`    | The size of the card.                                                              |
| borderRadius    | string  | `lg`    | The border radius of the card.                                                     |
| padding         | number  | `8`     | The padding of the card (in pixels).                                               |
| showDisplayName | boolean | `false` | If true, the display name will be shown.                                           |
| showAvatar      | boolean | `true`  | If true, the avatar will be shown.                                                 |
| bg              | boolean | `false` | If true, the card will have a background color.                                    |
| className       | string  | `""`    | Any additional [Tailwind](https://tailwindcss.com) classes to apply to the card.   |
| avatarProps     | object  | `{}`    | Any additional props to pass to the [`Avatar`](/docs/components/avatar) component. |
| ...props        | object  | `{}`    | Any additional props to pass to the &lt;div&gt; element.                           |

<Callout>
    Use `avatarProps` to pass additional props to the [`Avatar`](/docs/components/avatar) component.
</Callout>
