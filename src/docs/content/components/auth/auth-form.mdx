<div id="overview" />

<Caution>
    Requires [`Auth`](/docs/tutorial/authentication) to be set up.
</Caution>

The `AuthForm` component is a form component that handles user authentication. It can handle password-based, magic link,
and OAuth methods.

It creates `PasswordlessAuthForm`, `PasswordAuthForm`, and `OAuthButtons` components based on the methods available in the
[config.js](/docs/features/config#auth) file.

<Callout>
    The auth forms use [`ToastProvider`](/docs/components/toast) component to display toast messages.
</Callout>

<br/>
<br/>
<Divider />
<div id="usage" />
## Usage

```javascript
import AuthForm from '@/components/auth/AuthForm';

<AuthForm />
```

<Info>
    Runway uses the `AuthForm` component on the `/login` and `/signup` pages.
</Info>

<div id="props" />
## Props

| Name | Type   | Default     | Description                                          |
|------|--------|-------------|------------------------------------------------------|
| mode | string | `"sign-in"` | The mode of the form, can be `sign-in` or `sign-up`. |


<Info>
    It uses [`AuthInput`](/docs/components/auth/auth-input) and [`OAuthButton`](/docs/components/auth/oauth-button) components for the form fields and OAuth buttons respectively.
</Info>
