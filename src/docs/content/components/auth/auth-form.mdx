The `AuthForm` component is a form component that handles user authentication.

It **automatically retrieves** the authentication methods from the [config.js](/docs/features/config) file and displays the
appropriate form fields. It supports **email/password**, **magiclink** and **OAuth** authentication methods.

<Callout>
    It uses [`AuthInput`](/docs/components/auth/auth-input) and [`OAuthButton`](/docs/components/auth/oauth-button) components for the form fields and OAuth buttons respectively.
</Callout>
