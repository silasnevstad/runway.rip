<br/>

The `CheckoutButton` component is a button that initiates a checkout session with Stripe.
It's built on top of the [`Button`](/docs/components/button) component so you can customize it with all the same props.

<Callout>
    It calls the `/api/checkout_sessions` API route to create a [Stripe](https://stripe.com) checkout session.
</Callout>

<Caution>
    Requires [payments](/docs/tutorials/payments) to be configured.
</Caution>

## Usage

```jsx
import CheckoutButton from "@/components/payments/CheckoutButton";

<CheckoutButton
  planType="starter"
  intervalName="monthly"
>
    Subcribe Now
</CheckoutButton>

<CheckoutButton
  planType="pro"
  intervalName="one_time"
>
    Subcribe Now
</CheckoutButton>
```

## Props

| Name         | Type   | Default | Description                                                                      |
|--------------|--------|---------|----------------------------------------------------------------------------------|
| planType     | string | `""`    | The type of plan to subscribe to.                                                |
| intervalName | string | `""`    | The interval name for the plan.                                                  |
| ...props     | object | `""`    | All other props are passed to the [`Button`](/docs/components/button) component. |

<Tip>
    Since `CheckoutButton` is built on top of the [`Button`](/docs/components/button) component, you can pass any prop that it accepts to customize the button
    (i.e. `size`, `color`, `loading`, etc.).
</Tip>
