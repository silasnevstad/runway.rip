<br/>

The `Tooltip` component provides a lightweight and customizable tooltip that appears when hovering over an element.
It supports various positions and colors.

<br/>
<br/>
<Divider />

<div id="usage" />

## Usage

```javascript
import Tooltip from "@/components/atoms/Tooltip";

<Tooltip text="This is a tooltip">
    <div>
        Hover me
    </div>
</Tooltip>
```

## Props

| Prop      | Type                                   | Default     | Description                                                                             |
|-----------|----------------------------------------|-------------|-----------------------------------------------------------------------------------------|
| text      | string                                 | `""`        | The text to display inside the tooltip.                                                 |
| position  | "top" \| "bottom" \| "left" \| "right" | `"top"`     | The position of the tooltip relative to the target element.                             |
| color     | string                                 | `"primary"` | [Tailwind](https://tailwindcss.com) color key used for the toggle switch.               |
| className | string                                 | `""`        | Additional [Tailwind](https://tailwindcss.com) classes for the tooltip.                 |
| ...props  | object                                 | -           | Any other valid HTML <code>&lt;div&gt;</code> attributes (e.g. `role`, `aria-*`, etc.). |
