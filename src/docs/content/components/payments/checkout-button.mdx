<div id="overview" />

The `CheckoutButton` component is a button component that can be used to create a Stripe checkout session.

If you are using [Auth](/docs/features/authentication) and someone is logged in `CheckoutButton` will try to retrieve the customer ID from the user session.

<Callout>
    On click, the button sends a request to `/api/create-checkout` and redirects the user to the Stripe checkout page.
</Callout>

<Caution>
    `mode` ande `priceId` are always required.
    If you are doing subscriptions, `customerId` is also required.
</Caution>

## Usage

```javascript
import CheckoutButton from '@/components/payments/CheckoutButton';

<CheckoutButton
    mode="subscription"
    priceId="price_123"
    customerId="cus_123"
>
    Subcribe Now
</CheckoutButton>
```

<Info>
    [`PricingPlans`](/docs/components/payments/pricing-plans) uses the `CheckoutButton` component to create the checkout buttons.
</Info>

## Props

| Name          | Type   | Default | Description                                                                             |
|---------------|--------|---------|-----------------------------------------------------------------------------------------|
| mode          | string | `""`    | The mode of the checkout session (one of `subscription`, `payment`).                    |
| priceId       | string | `""`    | The price ID of the product.                                                            |
| customerId    | string | `""`    | The customer ID of the user.                                                            |
| customerEmail | string | `""`    | The email of the customer.                                                              |
| children      | node   | `""`    | The content of the button.                                                              |
| className     | string | `""`    | Additional [Tailwind CSS](https://tailwindcss.com/docs) classes to apply to the button. |
| ...props      | object | `{}`    | The other props to pass to the button.                                                  |
