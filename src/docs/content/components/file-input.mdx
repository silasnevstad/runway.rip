## Usage

```javascript
import FileInput from "@/components/atoms/FileInput";

<FileInput
    onFileChange={(file) => console.log(file)}
/>
```

## Props

| **Prop**        | **Type**  | **Default**                             | **Description**                                                                                           |
|-----------------|-----------|-----------------------------------------|-----------------------------------------------------------------------------------------------------------|
| text            | string    | `"Drop files here, or click to select"` | The text to display in the component.                                                                     |
| placeholder     | string    | `"No file selected"`                    | The placeholder text to display when no file is selected.                                                 |
| color           | string    | `"gray"`                                | The base color (use [tailwind.config](/docs/tutorials/config)).                                           |
| borderRadius    | string    | `"lg"`                                  | The border radius of the component (uses [Tailwind](https://tailwindcss.com/docs/border-radius) classes). |
| border          | boolean   | `true`                                  | If true, a border is displayed around the component.                                                      |
| icon            | ReactNode | `<FiFilePlus />`                        | The icon to display in the component.                                                                     |
| className       | string    | `""`                                    | Additional [Tailwind](https://tailwindcss.com) utility classes.                                           |
| buttonClassName | string    | `""`                                    | Additional [Tailwind](https://tailwindcss.com) utility classes for the button.                            |
| inputClassName  | string    | `""`                                    | Additional [Tailwind](https://tailwindcss.com) utility classes for the input.                             |
| onFileChange    | function  | -                                       | The function to call when a file is selected.                                                             |
| ...props        | object    | -                                       | Any other valid HTML attributes.                                                                          |
