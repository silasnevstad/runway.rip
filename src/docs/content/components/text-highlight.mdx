The `TextHighlight` component is a text component useful when needing to make some text stand out. It can be used to highlight text in a sentence or paragraph.

<Callout>
    You can either do a simple highlight or a gradient highlight.
</Callout>

## Usage

```js
import TextHighlight from "@/components/atoms/TextHighlight";

<TextHighlight highlight="highlight">
    This is a text highlight
</TextHighlight>
```

## Props

| Name         | Type   | Default         | Description                                                                                                        |
|--------------|--------|-----------------|--------------------------------------------------------------------------------------------------------------------|
| text         | string | -               | The text to display.                                                                                               |
| highlight    | string | -               | The text to highlight.                                                                                             |
| color        | string | `"gray"`        | The color of the text (see [tailwind.config](/docs/features/config#color-theme)).                                             |
| fromGradient | string | `""`            | The color of the gradient from which the text will be highlighted (see [tailwind.config](/docs/features/config#color-theme)). |
| toGradient   | string | `""`            | The color of the gradient to which the text will be highlighted (see [tailwind.config](/docs/features/config#color-theme)).   |
| textSize     | string | `"text-7xl/18"` | The text size and line height of the text.                                                                         |
| className    | string | `""`            | Additional [Tailwind](https://tailwindcss.com) classes for the container.                                          |
| ...props     | object | -               | Any other valid <code>&lt;a&gt;</code> attributes (e.g. `target`, `rel`, etc.).                                    |

## Examples

### Basic Text Highlight
You can customize the `color` and `textSize` of the text.

<PreviewCode preview={
    <div className="flex flex-col items-center gap-4">
        <TextHighlight text="Text highlight" highlight="Text highlight" color="red" textSize="text-5xl/10" />

}>
    ```js
    <TextHighlight
        text="Text highlight"
        highlight="Text highlight"
        color="red"

    />
    ```
