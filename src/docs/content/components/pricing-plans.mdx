<br />
The `PricingPlans` component is a simple and customizable pricing card that can be used to showcase a single pricing plan. It includes a **title**, **price**, **features**, and a **call-to-action button**.

<Callout>
    It uses the [`CheckoutButton`](/docs/components/checkout-button) component for the call-to-action button.
</Callout>

<Caution>
    You must configure [payments](/docs/tutorials/payments) first.
</Caution>

<br />
<br />

## Usage

```js
import PricingPlans from '@/components/payments/PricingPlans';

<PricingPlans />
```

<Info>
    It pulls everything it needs from [`pricingConfig`](/docs/features/config).
</Info>
