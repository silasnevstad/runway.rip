<br/>

The `Accordion` component displays content in collapsible panels.
It's useful for organizing information without overwhelming users.

<Callout>
    `Accordion` leverages [`DropdownText`](/docs/components/dropdown-text) to manage collapsible items.
</Callout>

<div id="usage" />

## Usage

```js
import Accordion from "@/components/atoms/Accordion";

<Accordion
    items={[
        {
            title: 'Overview Item 1',
            content: 'Content of item 1.'
        },
        {
            title: 'Overview Item 2',
            content: 'Content of item 2.',
            // You can also nest accordions
            items: [
                {
                    title: 'Nested Item 1',
                    content: 'This is a nested item.'
                },
                {
                    title: 'Nested Item 2',
                    content: 'This is another nested item.'
                }
            ]
        }
    ]}
/>
```

## Props

| **Name**  | **Type** | **Default** | **Description**                                                                         |
|-----------|----------|-------------|-----------------------------------------------------------------------------------------|
| items     | array    | -           | Array of objects, each containing `title` & `content` fields                            |
| oneOpen   | boolean  | `false`     | If true, only one accordion panel can be open at once                                   |
| borders   | boolean  | `true`      | If true, borders are applied to the accordion items                                     |
| level     | number   | `0`         | Used to automatically indent nested accordions                                          |
| className | string   | `""`        | Additional [Tailwind](https://tailwindcss.com) classes (overrides defaults)             |
| ...props  | object   | -           | Any other valid HTML <code>&lt;div&gt;</code> attributes (e.g. `role`, `aria-*`, etc.). |

<Tip>
    You can nest an Accordion inside another Accordion to create a multi-level collapsible structure for more complex documentation.
</Tip>

### Nested Accordion
Give an item an `items` prop to nest another accordion.

<PreviewCode preview={
    <div className="flex w-full px-2">
        <Accordion
            items={[
                {
                    title: "Nest Accordion",
                    content: "Content above the nested accordion:",
                    items: [
                        {
                            title: "Child Item",
                            content: "Content for nested item 1.",
                        },
                        {
                            title: "Another Nested Accordion",
                            items: [
                                {
                                    title: "Grandchild Item",
                                    content: "Content for grandchild item.",
                                },
                            ],
                        },
                    ],
                },
                {
                    title: "Simple Item",
                    content: "This is a simple accordion item.",
                },
            ]}
        />
    </div>
}>
    ```js
    <Accordion
        items={[
            {
                title: 'Parent Item 1',
                content: (
                    <Accordion
                        items={[
                            {
                                title: 'Child Item A',
                                content: 'Content of Child A.'
                            },
                            {
                                title: 'Child Item B',
                                content: 'Content of Child B.'
                            }
                        ]}
                    />
                )
            },
            {
                title: 'Parent Item 2',
                content: 'This is the second parent item content.'
            }
        ]}
    />
    ```
</PreviewCode>

<Caution>
    Be mindful of nesting too many layers of accordions. Deep nesting can confuse users or clutter your layout.
</Caution>
