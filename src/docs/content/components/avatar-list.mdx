<div>
    The `AvatarList` component groups together multiple avatars. You can show them in a row or column, stack them with overlap,
    and even limit how many are displayed (e.g., "+3 more").
</div>

<Callout>
    Each item is powered by the [Avatar](/docs/components/avatar) component, so you can customize it with any of its props.
</Callout>

## Usage

```javascript
import AvatarList from '@/components/molecules/AvatarList';

<AvatarList
    avatars={[
        { src: "/avatar1.png", alt: "User 1" },
        { letter: "B" },
        { src: "/avatar2.png", alt: "User 2" }
    ]}
/>
```

## Props

| **Prop**          | **Type**                   | **Default**    | **Description**                                                                       |
|-------------------|----------------------------|----------------|---------------------------------------------------------------------------------------|
| avatars           | array                      | `[]`           | Each object can contain `src`/`alt` or `letter`.                                      |
| direction         | "horizontal" \| "vertical" | `"horizontal"` | Chooses layout direction.                                                             |
| overlap           | boolean                    | `true`         | If true, applies negative margins (like Slack or GitHub) so the avatars stack.        |
| maxVisible        | number                     | `5`            | Max avatars displayed before the last is replaced with `+X more`.                     |
| sharedAvatarProps | object                     | `{}`           | Common props for every `Avatar` (e.g. `size="sm"`).                                   |
| className         | string                     | `""`           | Additional [Tailwind](https://tailwindcss.com) classes (overrides defaults if needed) |

<Tip>
    Use `sharedAvatarProps` to set common props for all avatars.
</Tip>

## Examples

### Horizontal List with Overflow
Set `maxVisible` to limit how many avatars are shown. The rest will be replaced with a "+X more" indicator.

<PreviewCode preview={
    <AvatarList
        avatars={[
            { src: "/avatar1.png", alt: "User A" },
            { letter: "B" },
            { src: "/avatar2.png", alt: "User C" },
            { src: "/avatar3.png", alt: "User D" },
            { letter: "X" },
            { letter: "Z" }
        ]}
        maxVisible={4}
    />
}>
    ```javascript
    <AvatarList
        avatars={[
            { src: "/avatar2.png", alt: "User A" },
            { letter: "B" },
            { src: "/avatar1.png", alt: "User C" },
            { src: "/avatar3.png", alt: "User D" },
            { letter: "X" },
            { letter: "Z" }
        ]}
        maxVisible={4}
    />
    ```
</PreviewCode>

<br />

### Vertical Stacking
Set `direction="vertical"` to stack avatars in a column. You can also adjust the avatar's size with `sharedAvatarProps`.

<PreviewCode preview={
    <AvatarList
        direction="vertical"
        avatars={[
            { letter: "A" },
            { src: "/avatar2.png", alt: "User B" },
            { letter: "C" }
        ]}
        sharedAvatarProps={{ size: "sm" }}
    />
}>
    ```javascript
    <AvatarList
        direction="vertical"
        avatars={[
            { letter: "A" },
            { src: "/avatar2.png", alt: "User B" },
            { letter: "C" }
        ]}
        sharedAvatarProps={{ size: "sm" }}
    />
    ```
</PreviewCode>
