The `TextLink` component is a text link component that can be used to navigate to other pages. It uses the [`Link`](https://nextjs.org/docs/app/api-reference/components/link) component from Next.js to create a link to another page. The `TextLink` component can be customized with different styles and options to fit the design of your application.

## Props

The `TextLink` component accepts the following props:

| Name               | Type      | Default | Description                                               |
|--------------------|-----------|---------|-----------------------------------------------------------|
| children           | ReactNode |         | The content of the link.                                  |
| href               | string    |         | The URL to link to.                                       |
| className          | string    |         | Additional classes to apply to the link.                  |
| icon               | Icon      |         | An icon to display next to the link text.                 |
| grow               | boolean   | `false` | Whether the link should grow to fill the available space. |
| fade               | boolean   | `false` | Whether the link should fade out on hover.                |
| underline          | boolean   | `false` | Whether the link should have an underline.                |
| underlineClassName | string    |         | Additional classes to apply to the underline.             |
| underlineHeight    | string    | `1px`   | The height of the underline.                              |
| selected           | boolean   | `false` | Whether the link is selected.                             |
