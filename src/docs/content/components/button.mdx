<div>
    The `Button` component can be customized with various colors, sizes, and shapes, has built-in loading states, hover effects,
    and supports icons. It can be rendered as either a `button` or a Next.js `Link` (if you provide the `href` prop).
</div>

## Usage

```jsx
import Button from "@/components/atoms/Button";

<Button
  color="primary"
  size="md"
  onClick={() => alert("Button clicked!")}
>
    Click Me
</Button>
```

## Props

| **Prop**     | **Type**             | **Default** | **Description**                                                                                                                 |
|--------------|----------------------|-------------|---------------------------------------------------------------------------------------------------------------------------------|
| children     | ReactNode            | -           | The content of the button.                                                                                                      |
| size         | "sm" \| "md" \| "lg" | `"md"`      | The size of the button.                                                                                                         |
| color        | string               | `"primary"` | The base color (uses [tailwind.config](/docs/features/config#color-theme)).                                                                 |
| borderRadius | string               | `"md"`      | The border radius of the button (uses [Tailwind](https://tailwindcss.com/docs/border-radius) border radius to shape component). |
| hoverBg      | boolean              | `true`      | If true, applies a hover background color.                                                                                      |
| scale        | boolean              | `false`     | If true, gently scales up on hover.                                                                                             |
| lift         | boolean              | `false`     | If true, lifts on hover.                                                                                                        |
| active       | boolean              | `true`      | If true, the button has an active press-down scale effect.                                                                      |
| border       | boolean              | `false`     | If true, adds a border.                                                                                                         |
| shadow       | boolean              | `false`     | If true, adds a subtle box-shadow.                                                                                              |
| loading      | boolean              | `false`     | If true, shows a spinner instead of children.                                                                                   |
| disabled     | boolean              | `false`     | If true, disables the button.                                                                                                   |
| icon         | ReactNode            | -           | If provided, the button will render an icon.                                                                                    |
| iconSrc      | string               | -           | If provided, the button will render an image as an icon.                                                                        |
| href         | string               | `""`        | If provided, the button becomes a `Link` instead of a `button`.                                                                 |
| onClick      | function             | -           | The click handler for the button.                                                                                               |
| className    | string               | `""`        | Additional [Tailwind](https://tailwindcss.com) utility classes.                                                                 |
| ...props     | object               | -           | Any other valid HTML attributes for the `button` or `Link`.                                                                     |

<br/>
<Info>
    Extend or override styles with the <code>className</code> prop if you need custom colors or advanced styling.
</Info>

## Examples

### Basic Usage
You can use the `Button` component like this:

<PreviewCode preview={

        <Button color={"red"} lift>Red + Lift Hover</Button>
        <Button size={"lg"} color={"blue"} scale>Large + Blue</Button>
        <Button color={"green"} variant={"soft"} borderRadius={"2xl"}>Green + Soft + Rounded</Button>
    </div>
}>
    ```js
    <div className="flex items-center flex-col gap-4">
        <Button size="sm" variant="soft" border>
            Small + Soft + Border
        </Button>
        <Button color="red" lift>
            Red + Lift Hover
        </Button>
        <Button size="lg" color="blue" scale>
            Large + Blue + Scale Hover
        </Button>
        <Button color="green" variant="soft" borderRadius="2xl">
            Green + Soft + Rounded
        </Button>
    </div>
    ```
</PreviewCode>

### Disabled & Loading
When `disabled` or `loading` is set, the button cannot be clicked, `loading` also shows a [`Loader`](/docs/components/loader).

<PreviewCode preview={

        <Button loading color="red">
            Loading...
        </Button>
    </div>
}>
    ```js
    <div className="flex flex-col gap-4">
        <Button disabled>
            Disabled
        </Button>
        <Button loading color="red">
            Loading...
        </Button>
    </div>
    ```
</PreviewCode>

<br />

### Icons
Pass a React icon component (`icon`) or an external image (`iconSrc`).

<PreviewCode preview={
    <ButtonIconDemo />
}>
    ```js
    <div className="flex flex-wrap gap-4 items-center">
        <Button
            icon={() => <ClipboardIcon className="w-5 h-5" />}
            className="px-5"
        >
            Copy
        </Button>
        <Button
            icon={() => <InformationCircleIcon className="w-5 h-5" />}
        >
            Click Me
        </Button>
    </div>
    ```
</PreviewCode>

<br />

### As a Link
If you pass an `href` prop, the button renders as a Next.js `Link`.

<PreviewCode preview={
    <div className="flex flex-wrap gap-4">
        <Button href="/docs" variant="underline">
            Go to Docs
        </Button>
    </div>
}>
    ```js
    <div className="flex flex-wrap gap-4">
        <Button href="/docs" variant="underline">
            Go to Docs
        </Button>
    </div>
    ```
</PreviewCode>

<Tip>
    Combine `href` with a `target="_blank"` for external links if desired.
</Tip>

<Caution>
    Avoid using `href` and `onClick` together if they lead to conflicting actions. `href` takes precedence.
</Caution>

<br />

### Custom Style
Using [Tailwind](https://tailwindcss.com) utility classes, you can create a custom button.

<PreviewCode preview={
    <div className="flex w-full px-2">

        >
            Custom Button
        </Button>
    </div>
}>
    ```js
    <Button
        className="bg-primary-900/20 dark:bg-primary-500/20 text-primary-500 hover:text-white"
    >
        Custom Button
    </Button>
    ```
</PreviewCode>
