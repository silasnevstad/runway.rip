<br />

The `Input` component creates a customizable text input field with various features like floating labels, left and right icons, and password visibility toggle.

<Tip>
    Use the `secure` prop to create a password input with a visibility toggle.
</Tip>

## Usage

```javascript
import Input from "@/components/atoms/Input";

<Input
  id="username"
  label="Username"
  placeholder="Enter your username"
/>
```

## Props

| Prop             | Type                         | Default                     | Description                                                                                                                                                                                                      |
|------------------|------------------------------|-----------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| id               | string                       | -                           | The id of the input.                                                                                                                                                                                             |
| label            | string                       | `""`                        | The label for the input.                                                                                                                                                                                         |
| labelMode        | "none" \| "above" \| "float" | `"none"`                    | If true, the label floats above the input when focused.                                                                                                                                                          |
| labelBackground  | string                       | `"bg-bg-50 dark:bg-bg-900"` | The background color of the label (only needed when using the `floatingLabel`). This is a little hacky, it will match the default background colors so you only need to modify when the background is different. |
| size             | "sm" \| "md" \| "lg"         | `"md"`                      | The size of the button.                                                                                                                                                                                          |
| borderRadius     | string                       | `"md"`                      | The border radius of the button (uses [Tailwind](https://tailwindcss.com/docs/border-radius) border radius to shape component).                                                                                  |
| shadow           | string                       | `""`                        | The shadow of the button (uses [Tailwind](https://tailwindcss.com/docs/box-shadow) shadow to shape component).                                                                                                   |
| leftIcon         | Icon                         | -                           | The icon to display on the left side of the input.                                                                                                                                                               |
| rightIcon        | Icon                         | -                           | The icon to display on the right side of the input.                                                                                                                                                              |
| leftIconOnClick  | function                     | -                           | The function to call when the left icon is clicked.                                                                                                                                                              |
| rightIconOnClick | function                     | -                           | The function to call when the right icon is clicked.                                                                                                                                                             |
| secure           | boolean                      | `false`                     | If true, the input is a password field with a visibility toggle (automatically adds a right icon to toggle visibility).                                                                                          |
| focus            | boolean                      | `true`                      | If true, applies a focus effect.                                                                                                                                                                                 |
| activeColor      | string                       | `"primary"`                 | The base color (use [tailwind.config](/docs/tutorials/config)).                                                                                                                                                  |
| value            | controlledValue              | -                           | The controlled value of the input.                                                                                                                                                                               |
| onChange         | function                     | -                           | The function to call when the input value changes.                                                                                                                                                               |
| className        | string                       | `""`                        | Additional [Tailwind](https://tailwindcss.com) utility classes.                                                                                                                                                  |
| textClassName    | string                       | `""`                        | Additional [Tailwind](https://tailwindcss.com) utility classes for the text.                                                                                                                                     |
| ...props         | object                       | -                           | Any other valid HTML attributes for the `button` or `Link`.                                                                                                                                                      |

<Callout>
    The component also accepts all standard HTML input attributes as additional props (i.e. `type`, `placeholder`, etc.).
</Callout>

<PreviewCode preview={

```javascript
import { PlusCircleIcon, PaperAirplaneIcon } from "@heroicons/react/24/outline";

<Input
    label="Left Icon"
    placeholder="Left Icon"
    labelMode="none"
    leftIcon={<PlusCircleIcon />}
```
