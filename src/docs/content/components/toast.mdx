<br />

The `Toast` component is a component that displays toast notifications. It is used to display messages, alerts, and other notifications to the user.

It has different `severity` levels such as `info`, `success`, `error`, and `warning`. It also has different styles such as `border`, `variant`, `scale`, etc.

<Callout>
    [`ToastProvider`](/docs/components/toast-provider) helps to manage the state of the toast notifications. It is used to display the toast notifications in a consistent manner.
</Callout>

## Usage

```js
import Toast from '@/components/atoms/Toast';

<Toast>
    This is a toast notification.
</Toast>
```

## Props

| Name        | Type                                        | Default   | Description                                                                                                                           |
|-------------|---------------------------------------------|-----------|---------------------------------------------------------------------------------------------------------------------------------------|
| children    | node                                        | -         | The content of the toast.                                                                                                             |
| variant     | "soft" \| "solid"                           | `"soft"`  | The variant of the toast.                                                                                                             |
| severity    | "info" \| "success" \| "warning" \| "error" | `"info"`  | The severity of the toast.                                                                                                            |
| border      | boolean                                     | `false`   | If true, the toast will have a border.                                                                                                |
| hoverBg     | boolean                                     | `true`    | If true, the toast will have a hover background.                                                                                      |
| active      | boolean                                     | `true`    | If true, the toast will have an active state.                                                                                         |
| scale       | boolean                                     | `false`   | If true, the toast will have a scale effect.                                                                                          |
| lift        | boolean                                     | `false`   | If true, the toast will have a lift effect.                                                                                           |
| autoDismiss | number                                      | -         | The time in milliseconds before the toast will be automatically dismissed. If not set, the toast will not be automatically dismissed. |
| closable    | boolean                                     | `true`    | If true, the toast will have a close button.                                                                                          |
| onClose     | function                                    | -         | The function to be called when the toast is closed.                                                                                   |
| onClick     | function                                    | -         | The function to be called when the toast is clicked.                                                                                  |
| slideFrom   | "left" \| "right"                           | `"right"` | The direction from which the toast will slide in.                                                                                     |
| className   | string                                      | `""`      | Any additional [Tailwind](https://tailwindcss.com) classes to apply to the toast.                                                     |
| ...props    | object                                      | -         | Any other valid <code>&lt;div&gt;</code> attributes.                                                                                  |

<Tip>
    The `autoDismiss` prop is used to automatically dismiss the toast after a certain amount of time. If not set, the toast will not be automatically dismissed.
</Tip>
