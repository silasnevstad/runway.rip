## Usage

Uncontrolled Usage
```javascript
import StarRating from "@/components/atoms/StarRating";

<StarRating rating={5}/>
```

<br/>

Controlled Usage
```javascript
import { useState } from "react";
import StarRating from "@/components/atoms/StarRating";

export default function FeedbackForm() {
    const [rating, setRating] = useState(3);

    return (
        <div>
            <h2>Please rate your experience:</h2>
            <StarRating
                totalStars={5}
                rating={rating}
                onRatingChange={setRating}
                responsive
                color="yellow"
            />
        </div>
    );
}
```

## Props

| **Prop**       | **Type**                    | **Default** | **Description**                                                                                                                      |
|----------------|-----------------------------|-------------|--------------------------------------------------------------------------------------------------------------------------------------|
| totalStars     | number                      | `5`         | The total number of stars to display.                                                                                                |
| rating         | number                      | `undefined` | The current rating if you want to control the component externally. If provided, the component becomes controlled.                   |
| onRatingChange | (newRating: number) => void | `undefined` | Callback that fires when the user selects a new rating. Use this to control the rating from outside the component.                   |
| responsive     | boolean                     | `false`     | Enables hover & click interactivity. If false, the stars will only reflect the static rating and will not respond to hover or click. |
| color          | string                      | `"yellow"`  | The base color for filled stars (uses [tailwind.config](/docs/tutorials/config#color-theme)).                                                    |
| initialRating  | number                      | `0`         | If the component is used in an uncontrolled fashion (no `rating` prop), this sets the initial rating.                                |
| className      | string                      | `""`        | Additional [Tailwind](https://tailwindcss.com) classes for the container.                                                            |
| ...props       | object                      | -           | Any other valid `<div>` attributes (e.g. `role`, `aria-*`, etc.).                                                                    |

<Callout>
    `StarRating` supports both <strong>controlled</strong> (via the <code>rating</code> and <code>onRatingChange</code> props) and <strong>uncontrolled</strong> usage (via the <code>initialRating</code> prop).
</Callout>

## Examples

### Basic Uncontrolled StarRating
To use `StarRating` in an uncontrolled fashion, simply provide the `initialRating` prop and set `responsive` to `false`.
You can also tweak the `totalStars` and `color` props to customize the component.

<PreviewCode
    preview={
        <div className="flex flex-col gap-4 mb-4 justify-center items-center">
            <StarRating totalStars={5} initialRating={2} responsive={false} />
        </div>
    }
>
    ```javascript
    <StarRating
        totalStars={5}
        initialRating={2}
        responsive={false}
    />
    ```
</PreviewCode>

<br />

### Fully Controlled StarRating

<PreviewCode
    preview={
        <div className="flex flex-col gap-4 mb-4 justify-center items-center">
            <StarRating
                totalStars={5}
                responsive
                rating={4}
            />
        </div>
    }
>
    ```javascript
    import { useState } from "react";
    import StarRating from "@/components/atoms/StarRating";

    function ControlledExample() {
        const [rating, setRating] = useState(4);

        return (
            <div>
                <StarRating
                    totalStars={5}
                    rating={rating}
                    onRatingChange={setRating}
                    responsive
                    color="red"
                />
                <p>Selected rating: {rating}</p>
            </div>
        );
    }

    export default ControlledExample;
    ```
</PreviewCode>
