The `StandoutCard`, `TerminalCard`, and `WebsiteCard` components are pre-styled cards that can be used to display content in a visually appealing way.

</Tip>

## StandoutCard
<br/>

<div className="flex flex-col items-center gap-4">
    <StandoutCard>
        <h2>Standout Card</h2>
        <p>This is a standout card.</p>
    </StandoutCard>
</div>

<br/>

The `StandoutCard` component is made to stand out with a unique design.

<br/>

```js
import { StandoutCard } from "@/components/molecules/CustomCards";

<StandoutCard>
    <h2>Standout Card</h2>
    <p>This is a standout card.</p>
</StandoutCard>
```

<br/>

### Props
| Prop           | Type      | Default     | Description                                                     |
|----------------|-----------|-------------|-----------------------------------------------------------------|
| children       | ReactNode | -           | The content of the card.                                        |
| borderRadius   | string    | `"2xl"`     | The border radius of the card.                                  |
| outerClassName | string    | `""`        | Additional [Tailwind](https://tailwindcss.com) utility classes. |
| innerClassName | string    | `""`        | Additional [Tailwind](https://tailwindcss.com) utility classes. |
| ...props       | object    | -           | Any other valid HTML attributes for the card.                   |

## TerminalCard
<br/>

<div className="flex flex-col items-center gap-4">
    <TerminalCard>
        <h2>Terminal Card</h2>
        <p>This is a terminal card.</p>
    </TerminalCard>
</div>

<br/>

The `TerminalCard` component is designed to resemble a terminal.

<br/>

```js
import { TerminalCard } from "@/components/atoms/CustomCards";

<TerminalCard>
    <span>Terminal Card</span>
    <p>This is a terminal card.</p>
</TerminalCard>
```

<Info>
    Wrap the first line in a <code>&lt;span&gt;</code> tag to make it part of the command line.
</Info>

<br/>

### Props
| Prop         | Type      | Default | Description                                                     |
|--------------|-----------|---------|-----------------------------------------------------------------|
| children     | ReactNode | -       | The content of the card.                                        |
| terminalText | string    | `"$ "`  | The text to display before the content.                         |
| className    | string    | `""`    | Additional [Tailwind](https://tailwindcss.com) utility classes. |
| ...props     | object    | -       | Any other valid HTML attributes for the card.                   |

## WebsiteCard
<br/>

<div className="flex flex-col items-center gap-4">
    <WebsiteCard>
        <h2>Website Card</h2>
        <p>This is a website card.</p>
    </WebsiteCard>
</div>

<br/>

The `WebsiteCard` component is styled to look like a website.

<br/>

```js
import { WebsiteCard } from "@/components/atoms/CustomCards";

<WebsiteCard>
    <span>Website Card</span>
    <p>This is a website card.</p>
</WebsiteCard>
```

<br/>

### Props
| Prop              | Type      | Default    | Description                                                     |
|-------------------|-----------|------------|-----------------------------------------------------------------|
| children          | ReactNode | -          | The content of the card.                                        |
| showSearchBar     | boolean   | `true`     | If true, shows a search bar.                                    |
| searchPlaceholder | string    | `"Search"` | The placeholder text for the search bar.                        |
| className         | string    | `""`       | Additional [Tailwind](https://tailwindcss.com) utility classes. |
| ...props          | object    | -          | Any other valid HTML attributes for the card.                   |
