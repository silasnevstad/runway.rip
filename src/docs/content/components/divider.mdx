<br />

The `Divider` component is a simple horizontal or vertical line that separates content. It can be customized with different colors, opacity levels, and text.

<Tip>
    Use the `opacity` prop to change the opacity of the divider (0-100).
</Tip>

## Usage

```javascript
import Divider from '@/components/atoms/Divider';

<Divider />
```

<Info>
    Use the `text` prop to add text to the divider.
</Info>

## Props

The Divider component accepts the following props:

| Name      | Type    | Default     | Description                                                                                      |
|-----------|---------|-------------|--------------------------------------------------------------------------------------------------|
| color     | string  | `"primary"` | The base color (use [tailwind.config](/docs/tutorials/config)).                                  |
| text      | string  | `""`        | Text to display in the divider.                                                                  |
| opacity   | number  | `100`       | Opacity of the divider (0-100).                                                                  |
| margin    | number  | `2`         | The margin around the divider (in px).                                                           |
| width     | number  | `0.5`       | The width of the divider (in rem).                                                               |
| height    | string  | `"full"`    | The height of the divider (uses [Tailwind](https://tailwindcss.com/docs/height) height classes). |
| vertical  | boolean | `false`     | If true, the divider will be vertical.                                                           |
| noWrap    | boolean | `true`      | If true, the text will not wrap.                                                                 |
| className | string  | `""`        | Additional [Tailwind](https://tailwindcss.com) utility classes.                                  |
| ...props  | object  | -           | Any other valid HTML attributes for the `button` or `Link`.                                      |
