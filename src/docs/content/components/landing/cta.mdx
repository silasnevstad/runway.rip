The `CTA` component is a call to action component that can be used to display a call to action on your website. It's minimal and perfect
great for landing pages.

<Info>
    It uses the [`Button`](/docs/components/button) component for the call to action button.
</Info>

## Usage

```javascript
import CTA from '@/components/landing/CTA';

<CTA />
```

## Props

| Name       | Type   | Default | Description                    |
|------------|--------|---------|--------------------------------|
| title      | string | `""`    | The title of the component.    |
| subtitle   | string | `""`    | The subtitle of the component. |
| buttonText | string | `""`    | The text of the button.        |
| buttonHref | string | `""`    | The href of the button.        |

<Callout>
    This component is already set up on `app/page.js` using [`landingConfig`](/docs/tutorials/config#landing-page).
</Callout>
