<div>
    The `Avatar` component displays a user’s profile image. If the image fails to load or isn’t provided, it
    automatically falls back to a letter. It supports various `sizes`, `shapes`, `colors`, and `status` indicators.
</div>

<Info>
    For multiple avatars, see the [AvatarList](/docs/components/avatar-list) component.
</Info>

## Usage

```js
import Avatar from '@/components/atoms/avatar';

<Avatar src="/avatar1.png" alt="User Image"/>
```

## Props

| **Prop**     | **Type**                             | **Default** | **Description**                                                                       |
|--------------|--------------------------------------|-------------|---------------------------------------------------------------------------------------|
| src          | string                               | -           | Image URL                                                                             |
| alt          | string                               | `""`        | Image alt text                                                                        |
| letter       | string                               | -           | Letter to display                                                                     |
| color        | string                               | `"gray"`    | Avatar color (from [tailwind.config](/docs/tutorial/config))                          |
| borderRadius | string                               | `"full"`    | The tailwind-friendly rounding for corners (e.g. `"md"`, `"full"`, etc)               |
| size         | "xs" \| "sm" \| "md" \| "lg" \| "xl" | `"md"`      | Avatar size                                                                           |
| border       | boolean                              | `true`      | Whether to show a border                                                              |
| status       | string                               | -           | A status indicator (e.g. 'online')                                                    |
| statusColor  | string                               | `"green"`   | Status color (from [tailwind.config](/docs/tutorial/config))                          |
| ringColor    | string                               | `"gray"`    | Border color when status is provided (from [tailwind.config](/docs/tutorial/config))  |
| onClick      | function                             | -           | Click handler                                                                         |
| className    | string                               | `""`        | Additional [Tailwind](https://tailwindcss.com) classes (overrides defaults if needed) |
| ...props     | object                               | -           | Additional props for the image element                                                |

<Tip>
    Feel free to customize the `status` prop any way you like, such as 'online' or 'busy'.
</Tip>

## Examples

### 1. Basic Image Avatar
Simply provide a `src` and `alt` prop. Optionally, style with other props.

<PreviewCode preview={
    <div className="flex gap-3">

            src="/avatar1.png"

            size="sm"

            alt="User Two"
            size="lg"
        />
    </div>
}>
    ```js
    <div className="flex gap-3">
        <Avatar

            alt="User One"
            size="sm"
        />
        <Avatar
            src="/avatar2.png"
            alt="User Two"
            size="lg"
        />
    </div>
    ```
</PreviewCode>

<br />

### 2. Letter
If you want a letter instead of an image, just provide the `letter` prop.

<PreviewCode preview={
    <div className="flex gap-3">
        <Avatar
            letter="Z"
            borderRadius="sm"
        />
        <Avatar
            alt="Ava Jones"
            src="/invalid-img.jpg"
        />
    </div>
}>
    ```js
    <div className="flex gap-3">
        <Avatar
            letter="Z"
            borderRadius="sm"
        />
        <Avatar
            alt="Ava Jones"
            src="/invalid-img.jpg"  // this will fail
        />
    </div>
    ```
</PreviewCode>

<Info>
    If the image is missing or broken, `Avatar` shows the the `letter` or the first character of the `alt` text.
</Info>

<br />
