<div>
    The `SearchBar` component allows users to search through a list of items that are organized into sections. The component filters both section titles and sub-items based on the search query and displays the results in a dropdown.
</div>

<Info>
    It leverages the [`Input`](/docs/components/input) component and the [`Modal`](/docs/components/modal) component when the `modalSearch` prop is set to `true`.
</Info>

## Usage

```javascript
import SearchBar from "@/components/atoms/SearchBar";

<SearchBar
    items={[
        {
            title: 'Item 1'
        },
        {
            title: 'Item 2'
        },
    ]}
    placeholder="Search docs..."
/>
```

## Props

| **Prop**           | **Type** | **Default**   | **Description**                                                                                                           |
|--------------------|----------|---------------|---------------------------------------------------------------------------------------------------------------------------|
| items              | array    | `[]`          | An array of objects representing the sections and items. Each object should have a `title` and an optional `items` array. |
| onItemSelect       | function | `() => {}`    | A callback function that is called when an item is selected. It receives the selected item as an argument.                |
| renderItem         | function | `null`        | A custom render function for each item. It receives the item as an argument and should return a React element.            |
| size               | string   | `"md"`        | The size of the input. Possible values are `sm`, `md`, `lg`.                                                              |
| placeholder        | string   | `"Search..."` | The placeholder text for the input.                                                                                       |
| modalSearch        | boolean  | `false`       | If `true`, the search will be performed in a modal.                                                                       |
| modalPlaceholder   | string   | `"Search..."` | The placeholder text for the modal input.                                                                                 |
| containerClassName | string   | `""`          | Additional [Tailwind](https://tailwindcss.com) classes for the container.                                                 |
| inputClassName     | string   | `""`          | Additional [Tailwind](https://tailwindcss.com) classes for the input.                                                     |
| textClassName      | string   | `""`          | Additional [Tailwind](https://tailwindcss.com) classes for the text.                                                      |
| ...props           | object   | -             | Any other valid <code>&lt;input&gt;</code>  attributes (e.g. `type`, `name`, etc.).                                       |

[//]: # (    sections,
[//]: # (    query,
[//]: # (    onSelectItem,
[//]: # (    renderItem,
[//]: # (    isModal,
[//]: # (    firstDisplayed,)

<Tip>
    The `items` prop should be an array of objects, where each object represents a section. Each section object should have a `title` and an optional `items` array. The `items` array should contain objects with `title` and `href` properties.
</Tip>
