<br />

The `Card` component is a flexible component that can be used to display content in a structured and visually appealing way.
It comes with animation and hover effects that can be customized to fit your design needs.

<Callout>
    Check out the [`CustomCards`](/docs/components/custom-cards) page for **pre-styled** custom cards.
</Callout>

## Usage

```javascript
import Card from "@/components/atoms/Card";

<Card>
    <h2>Card Title</h2>
    <p>This is a basic card with default settings.</p>
</Card>
```

## Props

| Prop         | Type                 | Default     | Description                                                     |
|--------------|----------------------|-------------|-----------------------------------------------------------------|
| children     | ReactNode            | -           | The content of the card.                                        |
| size         | "sm" \| "md" \| "lg" | `"md"`      | The size of the card.                                           |
| color        | string               | `"primary"` | The base color (use [tailwind.config](/docs/tutorials/config)). |
| variant      | "soft" \| "solid"    | `"solid"`   | The variant of the card.                                        |
| borderRadius | string               | `"2xl"`     | The border radius of the card.                                  |
| shadow       | boolean              | `false`     | If true, adds a subtle box-shadow.                              |
| hoverShadow  | boolean              | `false`     | If true, applies a hover shadow.                                |
| hoverBg      | boolean              | `false`     | If true, applies a hover background color.                      |
| scale        | boolean              | `false`     | If true, gently scales up on hover.                             |
| lift         | boolean              | `false`     | If true, lifts on hover.                                        |
| active       | boolean              | `false`     | If true, the card has an active press-down scale effect.        |
| outline      | boolean              | `false`     | If true, adds an outline effect on hover.                       |
| border       | boolean              | `false`     | If true, adds a border.                                         |
| href         | string               | `""`        | The URL to navigate to when the card is clicked.                |
| onClick      | function             | -           | The click handler for the card.                                 |
| className    | string               | `""`        | Additional [Tailwind](https://tailwindcss.com) utility classes. |
| ...props     | object               | -           | Any other valid HTML attributes for the card.                   |

<Info>
    Apply your own styling with the `className` prop for custom colors or advanced styling.
</Info>

### Card with Hover Effects
You can add and combine various hover effects using the `hoverBg`, `hoverShadow` `scale`, `lift`, and `outline` props.
You can also use `active` to enable the press-down scale effect.

<PreviewCode preview={
    <div className="flexl flex-col space-y-5 items-center">
        <Card hoverBg hoverShadow>
            <h2>Card with Background and Shadow</h2>
            <p>This card changes background color on hover.</p>
        </Card>

        <Card scale>
            <h2>Card with Scale Effect</h2>
            <p>This card scales up on hover.</p>
        </Card>

        <Card lift hoverShadow>
            <h2>Card with Lift + Shadow Effect</h2>
            <p>This card lifts on hover.</p>
        </Card>

        <Card outline>
            <h2>Card with Outline Effect</h2>
            <p>This card has an outline effect on hover.</p>
        </Card>

        <Card scale active>
            <h2>Card with Scale and Active Effect</h2>
            <p>This card scales up on hover and has a press-down effect.</p>
        </Card>
    </div>
}>
```javascript
<Card hoverBg>
    <h2>Card with Hover Background</h2>
    <p>This card changes background color on hover.</p>
</Card>

<Card scale>
    <h2>Card with Scale Effect</h2>
    <p>This card scales up on hover.</p>
</Card>

<Card lift>
    <h2>Card with Lift Effect</h2>
    <p>This card lifts on hover.</p>
</Card>

<Card outline>
    <h2>Card with Outline Effect</h2>
    <p>This card has an outline effect on hover.</p>
</Card>

<Card scale active>
    <h2>Card with Scale and Active Effect</h2>
    <p>This card scales up on hover and has a press-down effect.</p>
</Card>
```
</PreviewCode>

<Tip>
    The press-down effect will be enabled in `onClick` or `href` is provided. You can use `active` to enable it manually or disable it.
</Tip>
