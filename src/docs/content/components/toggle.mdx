<br />

The `Toggle` component is a simple switch that can be toggled on or off. It is commonly used to enable or disable a feature or setting.

<Tip>
    Use `showIcon` prop to display an icon inside the toggle switch.
</Tip>

## Usage

```javascript
import Toggle from "@/components/atoms/Toggle";

export default function Example() {
    const [isToggled, setIsToggled] = useState(false);

    return (
        <Toggle
            checked={isToggled}
            onChange={() => setIsToggled(!isToggled)}
        />
    );
}
```

## Props

The Toggle component accepts the following props:

| Name          | Type              | Default     | Description                                                                      |
|---------------|-------------------|-------------|----------------------------------------------------------------------------------|
| label         | string            | `""`        | The label to display next to the toggle switch.                                  |
| labelPosition | "left" \| "right" | `"left"`    | The position of the label relative to the toggle switch.                         |
| color         | string            | `"primary"` | [Tailwind](https://tailwindcss.com) color key used for the toggle switch.        |
| showIcon      | boolean           | `false`     | If `true`, an icon will be displayed inside the toggle switch.                   |
| focus         | boolean           | `false`     | If `true`, the toggle switch will be focused when rendered.                      |
| disabled      | boolean           | `false`     | If `true`, the toggle switch will be disabled.                                   |
| checked       | boolean           | `false`     | If `true`, the toggle switch will be in the "on" position.                       |
| onChange      | function          | -           | Callback function called when the toggle switch is toggled.                      |
| className     | string            | `""`        | Additional [Tailwind](https://tailwindcss.com) classes for the container.        |
| ...props      | object            | -           | Any other valid <code>&lt;input&gt;</code> attributes (e.g. `name`, `id`, etc.). |

<Info>
    The `onChange` prop is required to handle the toggle switch change event.
</Info>
